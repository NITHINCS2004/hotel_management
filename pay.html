Cost Calculation Function

The cost calculation function determines the total cost of a hotel stay based on several parameters:

It first calculates the number of nights between the check-in and check-out dates. Depending on the number of nights, the function applies different rates:

  • For the first 5 nights, it uses a base rate.
  • For the next 5 nights, it uses a slightly lower rate.
  • For any additional nights beyond the first 10, it uses a higher rate.

After calculating the cost for the nights, it multiplies this by the total number of guests (adults and children) to get the total guest cost.

Finally, it subtracts any paid cost (advance payment) from the total guest cost to determine the final cost.
